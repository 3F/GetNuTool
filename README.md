# [GetNuTool](https://github.com/3F/GetNuTool)

Embeddable Package Manager. NuGet / Chocolatey Client. And the lightweight portable tool for work with packages. *(via basic MSBuild that does not require any additional extensions, dotnet-cli, powershell, ...)*
